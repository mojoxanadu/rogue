### Merged (developer Kelch — Build 745 K)
Merged from Kelch's `745k3.txt`. Files: `state.js`, `ui_logic.js`, `engine.js`,
`map.js`, `quests_kq5.js`. Our `audio.js`, `input.js`, `header.html`, `ui_layout.html`
kept as newer.

#### Enhancements
1. **Generic enemies no longer move or attack diagonally** (`engine.js`). Enemies
   pick one axis (E/W or N/S) at random and only move along it. Adjacency check for
   attacks now requires same row or column.
2. **Randomized damage** (`engine.js`). Player and generic enemies now deal 1 to max
   damage instead of always max. Player melee: `rand(1..baseDmg) + meleeDmgBonus`.
3. **Direct stat system** (`state.js`, `ui_logic.js`). Removed player ability scores
   (str/dex/int/con/wis). Stats are now allocated directly: Max HP, Max MP,
   Melee/Ranged/Spell Damage Bonus, Hit Rate, Crit Rate, Dodge Rate. Damage bonuses
   cost 5 points per +1.
4. **`isTileFloor()` function** (`state.js`, `map.js`, `engine.js`, `quests_kq5.js`).
   Replaces most `=== TILES.FLOOR` checks with a generalized walkability check that
   includes FLOOR, GRASS, SAND, BRIDGE, BUSH.
5. **`setPlayerDefaults()` function** (`state.js`). Centralizes player reset logic
   used by both game start and restart.
6. **Tristram mapgen extracted** (`map.js`). `mapgenTown()` function replaces inline
   town generation in `initMap()`. Tristram tile map now uses string-array encoding.
7. **`enemyPassesThrough()` utility** (`engine.js`). Unified walkability check for
   enemies (handles throughWalls).

#### Fixes
1. **Ducks don't quack unless hit for damage** (`engine.js`). Quack sound moved inside
   the hit-success branch.
2. **Player can miss enemies, enemies can miss player** (`engine.js`). Hit chance is
   `player.hitRate * (1 - enemy.dodge)`. Monster attack uses same formula against player.
3. **Monster dodge values rebalanced** (`state.js`). Nearly all dodge values reduced
   (many to 0 or near-0) because dodge is now a meaningful bonus on top of hit rate.
4. **Initial player stats**: Hit rate 0.6 (was 0.50), crit rate 0.0 (was 0.10),
   dodge rate 0.0 (was 0.15). Initial HP 16 (was 25). New constants for all.
